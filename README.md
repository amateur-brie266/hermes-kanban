# 📋 hermes-kanban - Manage projects easily within Obsidian vaults

[![Download latest version](https://img.shields.io/badge/Download-Release-blue)](https://github.com/amateur-brie266/hermes-kanban/releases)

Hermes-kanban helps you track tasks and manage projects. It connects your Obsidian notes with the Hermes skill. This tool turns your Kanban boards into active project managers. You control your workflow without leaving your favorite note-taking app.

## ⚙️ System Requirements

This application works on Windows 10 and Windows 11. You need a modern computer with at least 8GB of RAM. The software performs best with an active internet connection. Please ensure you have the Obsidian application installed on your drive before you begin.

## 📥 Getting the Files

You must fetch the installation package from the official release page. Click the following link to view all available versions:

[https://github.com/amateur-brie266/hermes-kanban/releases](https://github.com/amateur-brie266/hermes-kanban/releases)

Choose the file ending in `.zip` or the installer package provided in the latest release section. Save this file to your Downloads folder.

## 🛠️ Installation Steps

Follow these steps to set up the plugin on your computer.

1. Locate the file you just downloaded.
2. Right-click the file and select "Extract All" if it is a compressed folder.
3. Open your Obsidian vault folder.
4. Locate the folder named `.obsidian`. If you cannot see it, enable "Hidden items" in your Windows File Explorer view settings.
5. Open the `plugins` folder inside `.obsidian`. Create this folder if it does not exist.
6. Move the extracted hermes-kanban folder into the `plugins` directory.
7. Restart the Obsidian application.

## 🔑 Activating the Plugin

After you restart Obsidian, activate the tool to start your work.

1. Open Obsidian.
2. Click the Settings icon in the bottom left corner.
3. Select "Community plugins" from the sidebar.
4. Click the "Installed plugins" section to find "hermes-kanban".
5. Toggle the switch to "On".
6. Navigate to the tool settings to enter your Hermes credentials if prompted.

## 📊 Using Kanban Boards

A Kanban board tracks your progress. You move cards between columns to show the status of your work.

### Create a board
To create a board, create a new note in your vault. Add the label "kanban" to the note properties. The plugin detects this label and changes the note into an interactive board.

### Add tasks
Click the plus icon at the top of any column. Type the name of your task and hit enter. These cards now link to your Hermes skill.

### Connect to Hermes
With the plugin active, your cards send signals to the Hermes system. If you move a card to the "Done" column, Hermes marks the task as complete in your project log. This keeps your records accurate and current.

## 🖱️ Managing Your Workflow

Effective project management relies on consistency. Keep your boards updated to maintain your project overview. 

You can create multiple boards for different projects. Name them clearly to help you find them later. The plugin index gathers all boards in the sidebar for quick access. 

If you delete a note, the plugin removes the associated board from the index. Your data remains safe inside the Obsidian note, even if you uninstall the extension.

## 💡 Troubleshooting

Follow these tips if you face issues while using the tool.

- Plugin not visible: Check the "Community plugins" menu. Ensure you enabled the plugin after installation.
- Boards do not load: Check your Obsidian version. Update Obsidian to the latest release to ensure full compatibility.
- Hermes sync errors: Check your internet connection. Verify that your Hermes skill account details are correct in the plugin settings menu.
- Slow performance: Delete old cards or archive boards that remain inactive. Keep your vault clean to improve search speeds.

## 🛡️ Data Privacy

Your data stays inside your vault. The plugin sends signals to Hermes to process tasks, but it does not store your notes on external servers. You keep full control over your content at all times.

## 📈 Improving Your Process

Use the different columns to build your own system. Most people use three columns: "To Do", "In Progress", and "Done". You can add more columns for specific needs, such as "Waiting" or "Review Needed".

The Hermes skill updates these columns based on your external work. It bridges the gap between your manual project management and automated system tasks.

## 🔄 Updating the Plugin

Your settings carry over automatically. You do not need to repeat the setup process during an update.
